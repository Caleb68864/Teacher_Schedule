Getting Started

First Time Setup
  1. Open your Google Sheet
  2. Click FlyOver Teaching → Launch Dashboard from the menu
  3. Grant permissions when prompted
  4. The app will create all necessary sheets automatically
  5. Start by adding students in the Students tab
Navigation
  • Schedule Tab: Manage class times and assignments
  • Students Tab: Student database and information
  • Analytics Tab: Charts and data visualization
  • Behavior Tab: Daily behavior tracking (1-3 scale)
  • Data Collection Tab: IEP data collection tools
  • Export Tab: Generate PDFs and backups

Schedule Management

Adding Schedule Entries
  1. Go to the Schedule tab
  2. Click "Add Entry"
  3. Fill in:
    • Day: Select day or day pattern (MWF, TTH)
    • Time: Enter time range (e.g., 9:00 AM - 10:00 AM)
    • Subject: Class or activity name
    • Students: Comma-separated list
    • Color & Icon: Visual customization
  4. Click "Save"
Schedule Types
  • Normal Regular classes
  • Special Special activities or events
  • Break Lunch, recess, prep periods

Student Management

Adding Students
  1. Navigate to Students tab
  2. Click "Add Student"
  3. Enter:
    • First and Last Name
    • Grade level
    • Email (optional)
    • Parent email (optional)
  4. Important: Check "Track behavior" for students needing behavior monitoring
  5. Click "Save"
Only students with "Track behavior" enabled will appear in behavior tracking and data collection tools.

Behavior Tracking

Daily Behavior Scoring

Use a 1-3 scale to track daily behavior:

  • 3 - Excellent Met or exceeded expectations
  • 2 - Good Generally met expectations with minor issues
  • 1 - Needs Improvement Significant challenges observed
Features
  • Weekly Grid: Quick entry for entire week
  • Color Coding: Visual feedback (Green/Yellow/Red)
  • Multiple Charts: Line, Bar, Radar, Scatter plots
  • Trend Analysis: Automatic detection of patterns
  • Streak Tracking: Count consecutive days of excellent behavior

Data Collection Tools

1. Frequency/Duration/Latency Recording

Track specific behaviors with detailed metrics:

  • Frequency: How many times the behavior occurred
  • Duration: How long the behavior lasted (minutes)
  • Latency: Time before response began (seconds)

Use for: Tracking specific behaviors like hand-raising, out-of-seat, verbal protests

2. ABC Tracking

Analyze behavior patterns using:

  • A - Antecedent: What happened before the behavior?
  • B - Behavior: What exactly did the student do?
  • C - Consequence: What happened after?

Automatic Features:

  • Function hypothesis suggestions
  • Pattern analysis
  • Intervention recommendations
3. Task Analysis

Break down complex skills into steps and track independence:

Prompt Levels (Most to Least Support):
FP - Full Physical PP - Partial Physical M - Model V - Verbal G - Gestural I - Independent
Pre-built Templates:
  • Hand Washing (7 steps)
  • Tying Shoes (8 steps)
  • Brushing Teeth (11 steps)
  • Putting on Coat (6 steps)
4. Work Sample Scoring

Track academic performance:

  • Score assignments by subject
  • Calculate percentages automatically
  • Add rubric criteria
  • Note accommodations used
  • Track trends across subjects
5. Prompt Level Tracking

Monitor prompt fading over time:

  • Track prompt level for specific skills
  • Record response latency
  • Measure accuracy percentage
  • Visualize fading progress
  • Identify when to fade prompts

IEP Progress Reports

Generating Reports
  1. Go to Data Collection tab
  2. Select a student
  3. Click "Generate IEP Progress Report"
  4. Report automatically compiles:
    • Behavior data and trends
    • Frequency/duration statistics
    • ABC analysis and recommendations
    • Task analysis progress
    • Academic performance
    • Prompt fading data
  5. Print or save the report
Reports include evidence-based recommendations based on data patterns!

Settings & Configuration

PDF Export Settings
  • Font Size: Base text size (default: 12pt)
  • Title Size: Header text size (default: 20pt)
  • Line Spacing: Space between lines (default: 1.5)
  • Margins: Page margins in mm (default: 15)
  • Orientation: Portrait or Landscape
  • Format: Letter or A4
Theme Customization
  • Title: Main app title
  • Colors: Primary and secondary theme colors
  • Schedule Colors: Default colors for different activity types
Behavior Settings
  • Chart Type: Default chart display
  • Alert Threshold: Score that triggers alerts
  • Good Threshold: Score for "good" behavior

Troubleshooting

Common Issues
Menu doesn't appear:
  • Refresh the Google Sheet
  • Check that Apps Script is enabled
  • Re-authorize when prompted
Data not loading:
  • Check sheet names match exactly
  • Use Settings → Initialize Sheets
  • Verify student has behavior tracking enabled
Charts not displaying:
  • Ensure data exists for the selected student
  • Check browser console for errors
  • Try refreshing the sidebar
PDF export fails:
  • Check browser allows popups
  • Verify data exists to export
  • Try different browser if issue persists
Getting Support
  • Check the documentation at the project repository
  • Review the CLAUDE.md file for technical details
  • Report issues on GitHub
  • Contact your IT administrator for deployment help

Pro Tips

Data Collection Best Practices
  • Be Consistent: Collect data at the same times each day
  • Be Specific: Use clear, observable behavior descriptions
  • Be Objective: Record what you see, not interpretations
  • Be Timely: Enter data as soon as possible
IEP Meeting Preparation
  1. Generate progress report 1 week before meeting
  2. Review ABC patterns for behavior insights
  3. Check task analysis for mastery levels
  4. Print work sample summary
  5. Prepare visual charts for parents
Keyboard Shortcuts
  • Tab - Navigate between fields
  • Enter - Save current entry
  • Esc - Cancel/close dialogs
