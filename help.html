Schedule Manager Help Guide

Quick Start

  1. View Schedule: Go to the main Schedule page to see your current schedule
  2. Edit Entries: Navigate to the Editor page to add, modify, or delete schedule entries
  3. Customize Appearance: Use the Theme Settings in the Editor to personalize colors and text
  4. Import/Export: Save your schedule as JSON or import from a file

Pro Tip

Your schedule is automatically saved to your browser's local storage. No need to manually save!

Schedule Viewer

The main schedule page displays all your time slots in an organized grid layout.

Features:

  • Refresh: Reload the schedule to see latest changes
  • Export JSON: Download your entire schedule as a JSON file for backup or sharing
  • Import JSON: Load a previously exported schedule file
  • Print: Generate a printer-friendly version of your schedule
  • Reset to Defaults: Restore the original demo schedule (WARNING: This erases all your changes!)
  • View Modes: Switch between Detailed (full info) and Compact (space-saving) views
  • Day Filter: Show only entries for specific days of the week

Schedule Card Elements:

  • Time Range: Shows start and end times with an icon
  • Subject: The main activity or class name
  • Grade Badge: Green badge showing the grade level
  • Students: Gray chips listing assigned students
  • Teachers: Blue chips showing assigned teachers
  • Notes: Yellow highlighted special instructions or reminders
  • Edit Button: Appears on hover - click to jump to editor

Schedule Editor

The editor provides comprehensive tools for managing your schedule entries.

Layout:

  • Left Panel: List of all schedule entries, theme settings, and student/teacher management
  • Right Panel: Detailed entry editor and live preview

Creating a New Entry:

  1. Click the green + New button at the top of the schedule list
  2. Fill in the required fields:
    • Start/End Time: Set the time range for this activity
    • Subject: Name of the class or activity
  3. Configure optional fields:
    • Grade: Grade level (e.g., "4th", "K-2", "Mixed")
    • Type: Choose Normal Class, Special Activity, or Break/Planning
    • Icon: Select a visual icon from the dropdown
    • Color: Pick a color theme for this entry
    • Students: Check boxes to assign students
    • Teachers: Check boxes to assign teachers
    • Note: Add any special instructions
    • Sort Order: Lower numbers appear first (default: 999)
    • Day of Week: Specify when this entry occurs
  4. Click Save Entry to add it to your schedule

Editing an Entry:

  1. Click on any entry in the left panel list
  2. The form will populate with current values
  3. Make your changes
  4. Click Save Entry to update

Deleting an Entry:

  1. Select the entry you want to remove
  2. Click the red Delete button
  3. Confirm the deletion when prompted

Warning

Deletions cannot be undone! Make sure to export your schedule regularly as a backup.

Color Customization

Each schedule entry can have its own color theme for easy visual identification.

Setting Entry Colors:

  • Click the color picker square to open a visual color selector
  • Or type/paste a hex color code (e.g., #3b82f6) in the text field
  • The preview will update in real-time

Suggested Color Meanings:

  • Blue (#3b82f6): Math and analytical subjects
  • Green (#10b981): Language arts and reading
  • Purple (#8b5cf6): Special programs
  • Pink (#ec4899): Social-emotional learning
  • Orange (#f59e0b): Breaks and planning time
  • Red (#ef4444): Important or urgent items

Student & Teacher Management

Managing Students:

  1. Click Manage Students in the editor's left panel
  2. To add: Type a name and click the green + button
  3. To remove: Click the red trash icon next to their name
  4. Students are automatically available as checkboxes when editing entries

Managing Teachers:

  1. Click Manage Teachers in the editor's left panel
  2. Add teachers with name, email, and phone (optional)
  3. Assign teachers to time slots just like students
  4. Teacher info helps with communication and scheduling

Organization Tip

Use consistent naming (First Last) for easy alphabetical sorting. For groups, use descriptive names like "PLC-4th Grade".

Theme Settings

Customize the overall appearance of your schedule viewer.

Available Settings:

  • Title: Main heading displayed on schedule page
  • Subtitle: Secondary text (e.g., academic year)
  • Background: CSS gradient or solid color for page background
  • Header Text Color: Color of title and subtitle text
  • Controls Background: Background color of the control panel

Example Backgrounds:

  • linear-gradient(135deg, #667eea 0%, #764ba2 100%) - Purple gradient
  • linear-gradient(90deg, #f093fb 0%, #f5576c 100%) - Pink sunset
  • #2d3748 - Solid dark gray

Import & Export

Exporting Your Schedule:

  1. Click Export JSON on the main schedule page
  2. Your browser will download a file named schedule-YYYY-MM-DD.json
  3. This file contains all entries, students, teachers, and theme settings

Importing a Schedule:

  1. Click Import JSON on the main schedule page
  2. Select a previously exported JSON file
  3. Confirm the import (this replaces your current schedule!)

Important

Importing a schedule will completely replace your current data. Always export a backup first!

Tips & Tricks

Best Practices:

  • Sort Order: Use multiples of 10 (10, 20, 30) to leave room for insertions
  • Day Filtering: Set "Tuesdays and Thursdays only" for recurring special activities
  • Icons: Choose icons that visually represent the subject for quick scanning
  • Notes: Include room numbers, special materials needed, or reminders
  • Backup: Export your schedule weekly or before major changes

Browser Compatibility:

This application works best in modern browsers:

  • Chrome/Edge (version 90+)
  • Firefox (version 88+)
  • Safari (version 14+)

Data Storage

How Your Data is Stored:

  • Local Storage: All data is saved in your browser's local storage
  • Privacy: No data is sent to external servers
  • Persistence: Data remains until you clear browser data or reset
  • Capacity: Can store hundreds of schedule entries

Privacy Note

Your schedule data never leaves your computer. Everything is stored locally in your browser for maximum privacy and security.

Troubleshooting

Common Issues:

  • Schedule not loading:
    • Try refreshing the page (F5)
    • Check if JavaScript is enabled in your browser
    • Clear browser cache and reload
  • Changes not saving:
    • Ensure you clicked "Save Entry" after making changes
    • Check if browser storage is full or disabled
    • Try exporting as backup, then reimporting
  • Print layout issues:
    • Use landscape orientation for best results
    • Switch to Compact View before printing
    • Adjust browser zoom to 100%

Reset Options:

  • Soft Reset: Click "Reset to Defaults" to restore demo data
  • Hard Reset: Clear browser data for this site in browser settings

Need More Help?

This schedule manager is designed to be intuitive and self-contained. If you encounter issues:

  1. Try exporting your data as a backup
  2. Refresh the page or restart your browser
  3. Check that you're using a supported modern browser
  4. As a last resort, use "Reset to Defaults" and re-import your backup
